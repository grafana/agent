# split

`split` produces a list by dividing a string at all occurrences of a separator.

```river
split(list, separator)
```

## Examples

```river
> split("foo,bar,baz", "," )
["foo", "bar", "baz"]

> split("foo", ",")
["foo"]

> split("", ",")
[""]
```
